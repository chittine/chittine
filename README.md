## Featured Projects

### Reliability-Aware Selective Gossip Protocol

Designed a distributed gossip protocol that combines reliability-aware peer selection with suppression mechanisms to reduce redundant message propagation while maintaining high delivery success rates across varying network conditions.

### Human Activity Recognition using Machine Learning

Developed a complete machine learning pipeline for activity recognition using dimensionality reduction, feature selection, Bayesian classifiers, SVM, Random Forest, and k-NN on high-dimensional sensor datasets.

### CUDA Matrix Multiplication Optimization

Optimized GPU matrix multiplication kernels through occupancy tuning, register tiling, tensor-core acceleration, and memory reuse techniques with detailed roofline-style performance analysis.

### GPU Run-Length Compression

Implemented high-throughput GPU-based compression and decompression using parallel scan, run detection, and scatter operations.

### Security Vulnerability Dashboard

Built a vulnerability analysis dashboard integrating automated OWASP ZAP scans, SQL-based reporting pipelines, and CI/CD automation for secure application monitoring.
